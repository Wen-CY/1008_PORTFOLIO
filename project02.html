小蜜搬家公司 Bee Move

Bootstrap HTML CSS 單頁式 點我前往該網站👇

網站運用技術：
運用Bootstrap框架，來創建網站。
快速創建出整體一致性，且擁有響應式設計的網站。


網站設計想法：
目的是讓來訪者，可以方便快速找到預約的方法。
所以運用單頁式網頁，無須跳轉至其他頁面，即可看到公司所提供的服務及價錢範圍。
加上多個CTA的按鈕，運用黃色醒目的強調色，方便顧客預約下單。
(目前馬上預約按鈕，暫無設置真正功能。)
